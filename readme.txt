QuestionPro Surveys plugin lets you embed surveys into the blog content.

== Description ==
Embedding QuestionPro surveys in the WordPress blogs has become easier than ever before!

QuestionPro Surveys plugin lets you embed QuestionPro surveys in your blogs by inserting a simple code snippet. You can also use this plugin to insert ready-made [QuestionPro Survey Templates](https://www.questionpro.com/survey-templates/) so that your readers can have view survey questions without having to click a link.

<strong>How to use QuestionPro Surveys plugin?</strong><br>
Download and install plugin on your blog. After successful installation, add below code snippet in your content and replace <Survey URL> with your survey url. Set the width and height of the frame, as appropriate.

[questionpro url="<Survey URL>" width="800" height="400"]

Example: [questionpro url="https://www.questionpro.com/a/TakeSurvey?tt=8jnRdJ5qGEU%3D" width="800" height="400"]


= Getting Started: =

Classic editor

*	Simply insert the above code, and you're good to go.

Block editor

*	 Click on "Add Block"
*  Select QuestionPro Surveys.
*  Add the Survey URL
*  Add the width and height, as required. This is optional.

1. Upload the entire `questionpro-surveys` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
